2. Cookie および同様のテクノロジーの使用方法

(1) クッキー

Web サイトが適切に機能することを保証するために、Cookie と呼ばれる小さなデータ ファイルがお客様のコンピュータまたはモバイル デバイスに保存されます。 Cookie には通常、識別子、サイト名、およびいくつかの数字と文字が含まれています。 Cookie を使用すると、Web サイトはユーザーの好みやショッピング バスケット内の商品などのデータを保存できます。

このポリシーに記載されている目的以外には Cookie を使用しません。好みに応じて Cookie を管理または削除できます。コンピュータに保存されているすべての Cookie をクリアできます。また、ほとんどの Web ブラウザには Cookie をブロックする機能があります。ただし、これを行う場合は、当社の Web サイトにアクセスするたびにユーザー設定をご自身で変更する必要があります。

(2) Webサイトビーコンとピクセルタグ

当社ウェブサイトでは、Cookie に加えて、Web ビーコンやピクセル タグなどの他の同様のテクノロジーも使用しています。たとえば、当社がお客様に送信した電子メールには、当社の Web サイト上のコンテンツにリンクされたクリック URL が含まれる場合があります。このリンクをクリックすると、お客様の製品またはサービスの好みを理解し、顧客サービスを向上させるために、このクリックが追跡されます。 Web ビーコンは通常、Web サイトまたは電子メールに埋め込まれた透明な画像です。電子メール内のピクセル タグの助けを借りて、電子メールが開封されたかどうかを知ることができます。この方法で自分の活動が追跡されることを望まない場合は、いつでもメーリング リストの購読を解除できます。

(3) 追跡しない

多くの Web ブラウザには、Web サイトに Do Not Track リクエストを発行する Do Not Track 機能があります。現在、主要なインターネット標準化団体には、Web サイトがそのようなリクエストにどのように対応するかを規定するポリシーがありません。ただし、ブラウザで [追跡拒否] が有効になっている場合、当社のすべてのサイトはお客様の選択を尊重します。

(1) 当社は、お客様から提供された個人情報を保護し、データを不正アクセス、公開、使用、改変、損傷または損失から防ぐために、業界標準に準拠したセキュリティ保護措置を講じています。当社は、お客様の個人情報を保護するためにあらゆる合理的な措置を講じます。たとえば、お客様のブラウザと「サービス」の間のデータ (クレジット カード情報など) の交換は SSL 暗号化によって保護されており、当社の Web サイトでは https による安全な閲覧も提供されており、データの機密性を確保します。データが悪意のある攻撃から保護されるよう、信頼できる保護メカニズムを使用します。アクセス制御メカニズムを導入して、許可された担当者のみが個人情報にアクセスできるようにします。また、セキュリティとプライバシー保護のトレーニング コースを開催して、従業員の重要性についての認識を強化します。個人情報を保護することを知っています。

(3) インターネットは完全に安全な環境ではないため、電子メール、インスタント メッセージング、および他のユーザーとの通信は暗号化されていません。そのような方法で個人情報を送信しないことを強くお勧めします。アカウントのセキュリティを確保するために、複雑なパスワードを使用してください。
